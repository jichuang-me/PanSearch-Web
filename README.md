# 🚀 PanSearch-Web: 极速云盘资源搜索

PanSearch-Web 是一个基于 Node.js 的全栈搜索应用，整合了多渠道 API 和实时链接验证，为您提供最快、最准的资源获取体验。

## ✨ 特性

- **极速响应**：支持本地 Node.js 后端代理或**纯前端 Serverless 模式**。
- **吸顶控制 (Sticky UI)**：搜索框与分类固顶，长页面操作更便捷。
- **点击标签筛选**：点击结果中的资源类型（如“电影”）即可一键联动筛选。
- **多引擎深度搜索**：实时并发抓取 **Baidu、Bing、Sogou、Google**，深挖网盘隐藏链接。
- **全网盘支持**：涵盖夸克、百度、阿里、UC、迅雷、移动、天翼、PikPak、115等。
- **GitHub Pages 适配**：无需服务器，直接在 GitHub Pages 或其他静态托管平台运行。

## 🛠️ 本地运行

如果您想在本地运行：

1. 克隆仓库：`git clone https://github.com/jichuang-me/PanSearch-Web.git`
2. 安装依赖：`npm install`
3. 启动：`npm start`
4. 访问：`http://localhost:8080`
